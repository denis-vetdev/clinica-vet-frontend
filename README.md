# 🐾 Clínica Vet — Frontend

Interface web para gerenciamento de clínica veterinária, consumindo a [Clínica Vet API](https://github.com/denis-vetdev/clinica-vet-api).

## 🚀 Tecnologias
- Angular 15
- Angular Material
- TypeScript
- RxJS
- JWT Authentication (integrado com API)

## ⚙️ Pré-requisitos
- [Node.js](https://nodejs.org/) v18+
- Angular CLI: `npm install -g @angular/cli`

## ⚙️ Como rodar localmente
```bash
git clone https://github.com/denis-vetdev/clinica-vet-frontend
cd clinica-vet-frontend
npm install
ng serve
```

Acesse `http://localhost:4200`

## 📸 Funcionalidades
- Login com autenticação JWT
- Dashboard com totais e próximas consultas
- CRUD de Tutores, Animais, Veterinários e Consultas
- Agendamento com calendário e seletor de horários
- Busca e filtros em todas as listagens
